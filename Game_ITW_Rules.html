Rules of Bejeweled

Candy Crush game Etapa 1 : Aparência - site base com navegação Organização: ficheiros HTML fora de pastas; pasta “media” para media ; pasta “style” para .css ; pasta “scripts” para .js ( html principal terá sempre de ser índex.html ) CSS - Posicionamento; cores e tipo de letra; decorações Etapa 2 (mínimos) : Candy Crash type beat, alinhar 3 ou mais jóias = pontos Individual = 1 jogo, 1 jogador ; dados de cada jogador guardados em local Storage do browser Tabuleiro 8x8 com 7 tipos de jóias diferentes, tempo de jogo, pontos, jóias eliminadas, botão para terminar Jogo : alinhar 3 jóias horizontalmente = eliminadas ; jóias que estavam por cima caem ; novas jóias aparecem aleatoriamente no topo Pontos: 1p = 3 jóias iguais ; pontos obtidos por alinhar jóias consecutivamente = (N-3) + 1 : 6 jóias == 2 pontos Mínimo de pontos = 18 e continua a escalar ate o fim do tempo
